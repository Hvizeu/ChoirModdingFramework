# Installation

1. Download the official Choir release for Songs of Syx `0.71.44`.
2. Extract it into the Songs of Syx user mod directory without nesting the
   `ChoirModdingFramework` folder twice.
3. Enable Choir in the game launcher or ChoirLauncher.
4. Keep Choir at the foundation of your profile. In ChoirLauncher, lower priority
   numbers load earlier, so Choir normally belongs at Priority `1`.
5. Keep every content mod that created rooms or other registry definitions enabled
   when loading saves that used it.

Choir is a separate dependency. Never embed its JAR inside another mod.
