# Security policy

Do not publish credentials, private keys, game saves, personal paths, or exploit
details in public issues. For a potentially security-sensitive report, contact
the project owner through the official Choir distribution page and include the
affected Choir version, Songs of Syx version, and minimal reproduction steps.

Unsupported game builds and fingerprint mismatches are compatibility reports,
not security vulnerabilities, but they are still useful when reported clearly.
